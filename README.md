## 📁 文件结构

```
├── data/                           # 数据文件夹
│   ├── iclr26_all_papers.json     # ICLR 2026所有投稿论文原始数据
│   ├── iclr26_ratings.json        # 论文评分数据
│   ├── iclr26_all_papers_with_ratings.json  # 整合评分后的完整数据
│   └── high_rated_papers.json     # 高评分论文数据（684篇）
├── add_ratings_to_papers.py       # 将评分数据添加到论文信息中
├── filter_high_rated_papers.py    # 筛选高评分论文
├── create_high_rated_html.py      # 生成高评分论文HTML展示页面
├── high_rated_papers.html         # 生成的HTML展示页面
├── high_rated_papers_summary.csv  # 高评分论文CSV摘要
├── fetch_ratings.py              # 获取评分数据（备用脚本）
├── request_iclr26.py             # 获取论文数据（备用脚本）
└── README.md                     # 项目说明文档
```
